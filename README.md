# meeknot.es
- Personal blog of Jonathan L. Meek

## Tech Stack 
- GitHub Pages 
- Powered by Hugo 
  - Uses the [hugo-theme-stack by CaiJimmy](https://github.com/CaiJimmy/hugo-theme-stack)

## Running Locally
- Git clone the repo
- Make sure to git clone submodules 
  - `git submodule update --init --recrusive`
- `hugo server`
- Naviagate to `http://localhost:1313` in a web browser
